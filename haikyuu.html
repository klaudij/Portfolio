Haikyuu!!

Jaar: 2021

Vak: Frontend Development

De opdracht

Voor het vak Frontend Development was de opdracht om 2 pagina’s van een bestaande website na te bouwen in vanilla HTML, CSS en Javascript. De website moest volledig responsive zijn of wat meer aandacht op de surface plane. Deze keuze lag helemaal aan jou.

Uiteindelijk heb ik mijn website zowel volledig responsive als aandacht voor de surface plane gekozen. Dit was eerst niet de bedoeling, want mijn voorkeur ging naar het volledig responsive maken, omdat ik wilde leren hoe dit nou precies moest. Maar in loop van de weken ben ik vanuit mijzelf (zonder er om na te denken) ook de surface plane meegnomen.

Ik heb al wat ervaring met HTML, CSS enJavascript, dus ik vond het namaken van de Officiële Haikyuu website hartstikke leuk!!

Ook al heb ik wat ervaring met coderen, ik heb zeker wat dingen geleerd tijdens de werkgroepen. Bijvoorbeeld het maken van een hamburger menu ment Javascript. Ik wist niet dat het zo makkelijk was. Natuurlijk heb ik ook geleerd hoe je een website volledig responsive maakt.
