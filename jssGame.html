Js game

Jaar: 2021

Vak: Inleiding programmeren

De opdracht

Voor het vak Inleiding Programmeren was de opdracht een game te maken met HTML, CSS en Javascript. Wat de game allemaal inhoudt mochten we zelfs beslissen als er maar verschillende Javascript elementen in voorkomen.

Ik heb ervoor gekozen om een Pokemon game te maken waarbij de bedoeling is dat je in het begon een starter Pokemon mag kiezen en hiermee op een camp zit.

Je kan jouw Pokemon laten vechten waardoor je geld verdient en tegelijkertijd de HP van de Pokemon verlaagd. Door op te tent te drukken of op de picnic deken verhoogt de HP. Het geld wat je opgespaard heb kan je uitgeven in de pokemon center, waarbij de bedoeling is dat je een nieuwe Pokemon kan kopen en die meeneemt naar je camp. Naast het onderhouden van je Pokemon kan je ook de current time zien, de achtergrond veranderen met een klik, de Pokemon laten praten door te klikken of slepen en kan hij over het scherm bewegen m.b.v. je toetsenbord.

Dit was de eerste keer dat ik volledig met Javascript heb gewerkt. Ik heb er al eerder mee wat gedaan, maar niet zoals bij dit vak. Dus ik heb enorm veel geleerd!
